# Rock Paper Scissors Game

## Overview

This is a simple Rock Paper Scissors game built using Python.

The user plays against the computer through the terminal. The computer randomly selects Rock, Paper, or Scissors, and the winner is decided based on the game rules.

---

# Features

- User vs Computer gameplay
- Random computer moves
- Input validation
- Replay option
- Beginner-friendly Python project

---

# Technologies Used

- Python
- Random module

---

# How to Run

## Clone the Repository

```bash
git clone https://github.com/yourusername/python-rock-paper-scissors.git
```

---

## Run the Game

```bash
python rock_paper_scissors.py
```

---

# Game Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
